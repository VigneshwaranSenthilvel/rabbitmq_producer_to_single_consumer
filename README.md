# EDUBOT-UI

AI chat application build in reactjs

## Dependencies

- [ASDF](https://asdf-vm.com/guide/getting-started.html)

## Run Locally

Clone the project

```bash
  git clone https://github.com/VigneshwaranSenthilvel/edubot_ui.git
```

Go to the project directory

```bash
  cd edubot_ui
```

Install dependencies

```bash
  asdf install
  npm install
```

Start the server

```bash
  npm run
```

## Demo in browser

http://localhost:3000
